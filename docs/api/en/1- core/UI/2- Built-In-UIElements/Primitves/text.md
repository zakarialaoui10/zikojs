  - ***`blockquote(...UIElement : []ZikoUIQuote)`*** : Creates a block quotation element with the provided ZikoUIQuote instances.

  
  **Usage**
   ```js

   const t2 = text("Hello from Zikojs").style({
    color: "darkblue"
   })
   const para = p(
    t1,
    t2
   ```
