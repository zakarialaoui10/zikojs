## Zikojs Layers

### `ziko-server` | Server Layer 
It provides Server-side rendering for Zikojs with file-based routing and client-side hydration.

#### Features : 
 - ✅ Server Side Rendering
 - ✅ Static Site Generation
 - ✅ Client Side Hydration with no additional config
 - ✅ File Based Routing
 - ✅ Dynamic Routing
 - ✅ Sync And Async rendering
 - ✅ API


### `mdzjs` | Content & Markup Layer 
A Markdown preprocessor for Zikojs.

#### Features

- **Extensible** : Create custom interactive components using Zikojs and use them in any Markdown file
- **Simple Integration**
- **Reusable**
- **Frontmatter Support**
- **JSX Syntax**
- **Props**
- **ESM**
- **Expressions**
- **Interleaving**
