**`ziko-server`** adds a *server-side rendering (SSR)* layer to *Zikojs*, it supports File based routing and client side hydration

## Features :

- ✅ Server Side Rendering
- ✅ Static Site Generation
- ✅ Client Side Hydration with no additional config
- ✅ File Based Routing
- ✅ Dynamic Routing
- ✅ Sync And Async rendering
- ✅ API
